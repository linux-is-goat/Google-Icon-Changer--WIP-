# Google Icon Changer

On a google website, the logo is mainly used in the websites favicon as well as in the top left of the screen.
Upon a simple assesment i found that the html controlling the top left icon on google sites is:
```<img class="gb_6c gb_ge" src="https://ICON_HERE.com" srcset="https://ICON_HERE.com 1x, https://ICON_HERE.com 2x" alt aria-hidden="true" role="presentation" style="width:40px;height:40px">```  and i have to find a way to change it to my preffered icons
